## üí° Key Insights

### Innovation Type: Sustaining

Vertech primarily invests in **sustaining innovation**:

- Enhancing existing automation architectures  
- Improving UI/UX for operators and engineers  
- Making plants more connected, efficient, and reliable  

This positions them as a trusted, long-term partner for large industrial clients rather than a disruptive startup.

### IP Strategy

Vertech protects its innovation through:

- **Trademarks** for logos and branded Ignition solutions  
- **Copyrights** on software and engineering assets  
- A **U.S. patent** for an Electro-Acoustic Resonance Heater, demonstrating applied R&D capability

   - Auto-suggest architectures, components, and integration patterns  
   - Reduce proposal & engineering time

2. **Interactive Solution Previews**  
   - Auto-generate mock HMI/SCADA views or process diagrams  
   - Help customers ‚Äúsee‚Äù the value before committing

3. **Business Model Evolution**  
   - Offer trials, pilots, and tiered subscription packages  
   - Create recurring revenue beyond one-time project delivery
